# Enhanced Clipboard

[English](./README.md)

一个自用 Windows 剪贴板小工具，顺手开源出来，也许对别人也刚好有点用。

目标很简单：让剪贴板历史比 Windows 自带的体验更实用一点，但不把它做成一个臃肿的“全能生产力平台”。

它会保持聚焦，带一点个人取向，也会按照我自己的节奏继续开发。

---

## 这是什么

Enhanced Clipboard 是一个基于 **Tauri**、**Rust**、**Vue 3** 和 **TypeScript** 构建的桌面剪贴板管理器。

它首先是一个**个人使用项目**。我自己在用，也会持续改进那些我真正在意的部分；之所以公开到 GitHub，只是因为它或许也能帮到其他人。

这也意味着，它的开发方向更多来自真实使用需求，而不是去覆盖所有可能的场景。

---

## 为什么做它

Windows 自带的剪贴板历史并不差，但对我的日常使用来说，还是稍微有点不够。

我想要一个更顺手的小工具，能更方便地：

- 浏览最近复制过的内容
- 搜索之前复制过的条目
- 按日期筛选
- 置顶常用内容
- 快速把文本或图片重新复制回剪贴板

所以我就自己做了一个。

---

## 功能

- 文本和图片的剪贴板历史
- 关键词搜索
- 使用斜杠命令菜单做类型筛选
- 日期筛选
- 置顶条目
- 文本条目的轻量语义标签
- 已保存内容一键复制回剪贴板
- 本地存储
- 一些实用但克制的设置项

没有特别花哨的东西，基本都是日常真正会用到的功能。

---

## 技术栈

- **Tauri v2**
- **Rust**
- **Vue 3**
- **TypeScript**
- **Pinia**
- **SQLite / SQLCipher**

---

## 项目范围

这个项目会尽量保持小而聚焦。

它**不是**为了做成：

- 云同步服务
- 团队协作工具
- 笔记软件
- “什么都想做”的剪贴板平台

它只是一个本地桌面剪贴板管理器，重点是把使用流程做得更干净、更实用。

---

## 项目状态

这个项目已经能用，而且我自己也一直在用，但它本质上仍然首先是一个个人项目。

有几点可以提前说清楚：

- 更新频率可能不固定
- bug 不保证会立刻修复
- 是否增加新功能，主要取决于它是否符合项目方向，以及我自己是否真的需要
- 不太会为了“让更多人都能用”而不断扩张功能边界

欢迎提 issue 和 pull request，但这个仓库并不是那种高频维护、快速响应的社区型项目。

## 开发

### 克隆仓库

```bash
git clone https://github.com/kuonji-arisu/enhanced-clipboard.git
cd enhanced-clipboard
````

### 安装依赖

```bash
pnpm install
```

### 启动开发模式

```bash
pnpm tauri dev
```

### 构建

```bash
pnpm tauri build
```

### 测试

```bash
pnpm test
```

如果只想单独跑前端或后端测试，也可以：

```bash
pnpm test:frontend
pnpm test:backend
```
